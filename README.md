# Poutro

## Quick Start

Install [stack](https://docs.haskellstack.org/en/stable/README/)

```console
$ nix-shell          # if you are using NixOS
$ stack build
$ mkdir out/
$ stack exec poutro names.txt out/
$ ./render-video.sh
```
